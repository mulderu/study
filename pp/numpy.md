## dot out production 

```python
>>> a=np.array([1,2,3,4])
>>> b=np.array([0.1,2,1])
>>> a=np.reshape(a, len(a), 1)
>>> b=np.reshape(b,(1,len(b)))
>>> a
array([[1],
       [2],
       [3],
       [4]])
>>> b
array([[0, 1, 2, 1]])
>>> a.shape
(4, 1)
>>> b.shape
(1, 4)
>>> c=a*b
>>> c
array([[0, 1, 2, 1],
       [0, 2, 4, 2],
       [0, 3, 6, 3],
       [0, 4, 8, 4]])
>>> a[:2]
array([[1],
       [2]])
>>> c[:2]
array([[0, 1, 2, 1],
       [0, 2, 4, 2]])
>>> c[0:2]
array([[0, 1, 2, 1],
       [0, 2, 4, 2]])
>>> c[0,2]
2
>>> c[-1]
array([0, 4, 8, 4])
>>> c[:-1]
array([[0, 1, 2, 1],
       [0, 2, 4, 2],
       [0, 3, 6, 3]])
>>> c[-2]
array([0, 3, 6, 3])
>>> c[:-2]
array([[0, 1, 2, 1],
       [0, 2, 4, 2]])
>>> c[2,3]
```
